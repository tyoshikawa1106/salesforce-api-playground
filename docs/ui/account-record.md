# 取引先詳細

## アクセス方法

- URL は `/accounts/<取引先ID>`。
- 取引先一覧で取引先名を選択する。
- ヘッダー検索で取引先を選択する。
- 取引先責任者詳細の取引先リンクから移動する。
- ブラウザの戻る / 進むで、直前の一覧または詳細へ戻れる。

## 表示内容

- 取引先名。
- 種別、電話、Web サイト、業種、請求先。
- 詳細情報。
- 作成日、最終更新日。
- 活動。
- 関連する取引先責任者。

## 操作

- 取引先情報を更新できる。
- 取引先を編集できる。
- 取引先を削除できる。
- 活動を確認できる。
- 電話、ToDo、行動を追加できる。
- 関連する取引先責任者を開ける。
