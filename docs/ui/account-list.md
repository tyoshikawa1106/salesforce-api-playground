# 取引先一覧

## アクセス方法

- URL は `/accounts`。
- Salesforce 接続後、上部ナビゲーションの `取引先` を選択する。
- 取引先詳細を開いている場合は、上部ナビゲーションの `取引先` を選択し直すと一覧へ戻る。
- ブラウザの戻る / 進むで、直前の一覧または詳細へ戻れる。

## 表示内容

- 取引先の一覧。
- 取引先名、電話、Web サイト、業種、請求先、最終更新日、最終更新者。
- 表示件数。
- リスト内検索欄。
- 一覧が空の場合のメッセージ。
- 読み込み中の小さいスピナー。

## 操作

- 取引先を新規作成できる。
- 一覧を更新できる。
- リスト内検索で表示中の取引先を絞り込める。
- 取引先名を選択して取引先詳細を開ける。
- 行ごとに編集または削除できる。
- 複数の取引先を選択して一括削除できる。
