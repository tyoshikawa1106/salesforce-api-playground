# 取引先責任者詳細

## アクセス方法

- URL は `/contacts/<取引先責任者ID>`。
- 取引先責任者一覧で氏名を選択する。
- ヘッダー検索で取引先責任者を選択する。
- 取引先詳細の関連取引先責任者から移動する。
- ブラウザの戻る / 進むで、直前の一覧または詳細へ戻れる。

## 表示内容

- 取引先責任者名。
- 取引先名、役職、部署、メール、電話。
- 詳細情報。
- 作成日、最終更新日。
- 活動。

## 操作

- 取引先責任者情報を更新できる。
- 取引先責任者を編集できる。
- 取引先責任者を削除できる。
- 関連する取引先を開ける。
- 活動を確認できる。
- 電話、ToDo、行動を追加できる。
