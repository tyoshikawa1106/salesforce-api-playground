# ホーム

## アクセス方法

- Salesforce 接続後、上部ナビゲーションの `ホーム` を選択する。
- 接続確認中は、ホームではなく接続状態の確認画面が表示される。
- 未接続の場合は、Salesforce 接続画面が表示される。

## 表示内容

- `Salesforce API Playground` の画面見出し。
- アプリの概要。
- Salesforce 接続状態。
- 接続先インスタンス。
- 接続済み画面では、上部ヘッダー、検索欄、主要ナビゲーション。

## 操作

- 主要ナビゲーションから各画面へ移動できる。
- 接続済みの場合は、ヘッダーの検索欄から取引先または取引先責任者を検索できる。
- ログアウトできる。
